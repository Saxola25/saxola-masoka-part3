# Sea Friendly Website

## Project Title
Sea Friendly – Marine Conservation Awareness Website

## Project Description
Sea Friendly is an educational website developed to promote awareness about marine conservation and environmental sustainability. The website provides information about protecting oceans, the services offered by the organisation, and ways for visitors to get involved through enquiries and volunteering.

The website was designed to be user-friendly, responsive, and accessible across different devices.

---

## Project Objectives

The objectives of this website are to:

- Educate users about marine conservation.
- Promote environmental awareness.
- Encourage community involvement.
- Provide information about Sea Friendly services.
- Allow users to contact the organisation through online forms.

---

## Website Features

### Navigation Menu
A responsive navigation menu allows users to move easily between pages.

### Home Page
Provides an introduction to Sea Friendly and its mission.

### About Us Page
Explains the organisation's background, vision, and goals.

### Services Page
Displays the services and environmental initiatives offered.

### Contact Page
Provides contact information and communication options.

### Enquiry Form
Allows visitors to submit enquiries directly through the website.

### EmailJS Integration
The enquiry form uses EmailJS to send messages without requiring a backend server.

### Responsive Design
The website adapts to desktops, tablets, and mobile devices.
